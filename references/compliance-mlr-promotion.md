# 推广合规与MLR / 合规体系与培训赋能（维C1·C7）

> 本 reference 给合规官一份"推广材料审查 + 合规培训内容产出"手册。**合规课件由本 Skill 产出，转译交付 drug-training 做课件与考核。**

## C1 推广合规与MLR审查
### C1.1 推广材料分级
| 级 | 类型 | 审查强度 |
|---|---|---|
| A | 医生告知书(DA)、核心Slide | 全文 MLR，证据溯源 |
| B | 讲者幻灯、会议资料 | 全文 MLR + 讲者资质 |
| C | 销售话术卡、Q&A | MLR 要点 + 禁表述清单 |
| D | 内部培训材料 | 内容合规（超说明书/贬低竞品） |

### C1.2 MLR 审查清单（逐项）
- [ ] 未超说明书适应症/用法（无超适应症推广）
- [ ] 无对比性贬低竞品（"更安全/更好"须有头对头证据）
- [ ] 数据可溯源（研究名/年份/样本量/终点，标 🔍）
- [ ] 不良反应信息完整（不隐匿已知 ADR）
- [ ] 无疗效/安全性断言或治愈率保证（接广告法 C3）
- [ ] 利益冲突声明（讲者/顾问）
- [ ] 合规承诺签署（接 drug-sales S6 / drug-training 合规课件）

### C1.3 审查结论
- **通过**：可发布；**修改**：列明条款退回；**否决**：违规表述删除前不得用。

## C7 合规体系与培训赋能
### C7.1 合规制度框架
- 推广合规管理办法、反商业贿赂制度、广告审查办法、数据隐私办法、举报与问责机制。

### C7.2 合规培训内容产出（→ drug-training）
- 模块：医药代表九大禁止、CSO 合作红线、礼品款待标准、学术活动合规、广告法要点、数据隐私、案例警示。
- **本 Skill 产出内容与红线话术；drug-training 负责转译为新人训/在职训课件 + 考核（柯氏 L3 行为闭环归 drug-hr H6/H8）**。

### C7.3 合规考核闭环
- 合规承诺必签（新人训必过项）；行为抽查（接 drug-sales S7 预警）；违规问责接 drug-hr H8。
