# 专利FTO与IP布局 / 监管应对（维C5·C6 · ⭐事实锚定）

> 本 reference 给法务/IP 角色一份"FTO 与专利保护"落地手册。专利号/到期日须以 CNIPA/USPTO/智慧芽为准 🔍，禁凭记忆。基础框架见 drug-base §5。

## C5 专利FTO与IP布局（事实锚定 🔍）
### C5.1 中国药品专利四重保护架构
据《专利法》（2020修正）与《药品专利纠纷早期解决机制实施办法（试行）》（2021），中国形成四重保护 🔍：
1. **专利保护**：发明专利 20 年 + 专利期限补偿（PTE）最长 5 年，但上市后有效专利期不超过 14 年 🔍。
2. **专利链接**：仿制药申请人就所列专利作四类声明；**第四类声明**（挑战专利有效性或主张不侵权）触发 **45 天**诉讼起始窗口；化学仿制药面临 **9 个月**审批等待期；生物类似药**不适用**等待期 🔍。
3. **数据保护**：最长 6 年 🔍。
4. **市场独占**：2—7 年（首仿挑战成功可获 12 个月市场独占）🔍。

### C5.2 FTO（自由实施）分析
- **节点**：立项前、临床前、上市前、融资/并购/license 交易等关键节点必做 🔍。
- **方法**：确定目标药技术特征 → 按"全面覆盖原则"和"等同原则"比对高度相关专利权利要求 → 判断是否落入保护范围 🔍。
- **应对**：高风险专利可提无效请求/第三方公众意见/规避设计；关注在审专利授权文本变动，动态复评 🔍。

### C5.3 Bolar 例外
《专利法》第75条第5项：为获取药品行政审批所需信息，制造、使用、进口专利药品（含向仿制药企供应原料）不视为侵权 🔍。

### C5.4 无效攻防与布局
- 核心专利布局：化合物/晶型/制备方法/用途专利组合。
- 无效理由常见：公开不充分、权利要求不清楚/不支持、缺乏新颖性/创造性 🔍。
- 接 drug-bd：交易 IP 审查（FTO 为 DD 必查项）、NewCo/license 的 IP 归属与地域。

## C6 监管应对与危机
- **飞检/稽查预案**：资质（生产/经营许可、GSP/GMP）、台账、推广记录、费用凭证完好可查。
- **投诉与不良反应**：按《药品不良反应报告和监测管理办法》上报，接 drug-reg。
- **危机分级**：按影响（监管函/媒体/召回）定发声口径与上报路径；对外口径须经合规+法务。
- **跨境**：出海业务须查目标国出口管制（如美国 EAR）、数据合规（GDPR/人类遗传资源）🔍，接 drug-bd 出海。
