# 广告与营销合规（维C3 · ⭐事实锚定）

> 本 reference 给合规官一份"药品/医疗器械/保健食品/特医食品广告审查"落地手册。法规标 🔍，不确定标 💭。**未经审查不得发布药品广告** 🔍。

## C3.1 法规依据
- 《中华人民共和国广告法》（2018修正）第十五条、第十六条、第十七条、第十八条、第十九条 🔍。
- 国家市场监管总局**21 号令**《药品、医疗器械、保健食品、特殊医学用途配方食品广告审查管理暂行办法》（2020-03-01 施行）🔍。
- 发布前须由广告审查机关审查；**未经审查不得发布** 🔍（《广告法》第四十六条）。

## C3.2 《广告法》第15条：特殊药品与处方药媒介限制 🔍
- 麻醉药品、精神药品、医疗用毒性药品、放射性药品等特殊药品，药品类易制毒化学品，以及戒毒治疗的药品/医疗器械/治疗方法，**不得作广告** 🔍。
- 前款规定以外的**处方药**，只能在国务院卫生行政部门和药品监督管理部门共同指定的**医学、药学专业刊物**上作广告 🔍。
- 《广告法》第16条与第18条区别：第16条管"医疗、药品、医疗器械广告"的禁止内容；**第18条管"保健食品广告"**（不得涉疾病预防治疗功能、不得与药品比较、不得用代言人等）🔍。二者禁止清单不同，审查时按产品属性分别适用 💭（具体边界以最新法条为准）。

## C3.3 《广告法》第16条：药品广告五项禁止 + 标示义务 🔍
医疗、药品、医疗器械广告**不得含有**下列内容：
1. 表示功效、安全性的**断言或者保证**；
2. **说明治愈率或者有效率**；
3. 与**其他药品、医疗器械的功效和安全性或者其他医疗机构比较**；
4. 利用**广告代言人**作推荐、证明；
5. 法律、行政法规规定禁止的其他内容。

**标示义务**（药品广告）：
- 内容不得与国务院药监部门批准的**说明书不一致** 🔍；
- 应当显著标明**禁忌、不良反应** 🔍；
- **处方药广告**应当显著标明"**本广告仅供医学药学专业人士阅读**" 🔍；
- **非处方药广告**应当显著标明"**请按药品说明书或者在药师指导下购买和使用**" 🔍。

## C3.4 21 号令核心：第11条 八项禁止情形 🔍
药品、医疗器械、保健食品和特医食品广告不得违反《广告法》第九、十六、十七、十八、十九条，**不得包含下列情形**（21号令第11条）🔍：
1. 使用或变相使用国家机关、国家机关工作人员、军队单位/人员名义或形象，或利用军队装备设施从事广告宣传；
2. 使用科研单位、学术机构、行业协会或专家、学者、医师、药师、临床营养师、患者等名义或形象作推荐、证明；
3. 违反科学规律，明示或暗示可治疗所有疾病、适应所有症状/人群，或正常生活和治疗病症所必需等内容；
4. 引起公众对健康/疾病不必要的担忧恐惧，或使公众误解不使用会患病/加重病情的内容；
5. 含有"安全""安全无毒副作用""毒副作用小"；明示或暗示成分为"天然"因而安全性有保证等内容；
6. 含有"热销、抢购、试用""家庭必备、免费治疗、免费赠送"等诱导性内容，"评比、排序、推荐、指定、选用、获奖"等综合性评价内容，"无效退款、保险公司保险"等保证性内容，怂恿任意/过量使用的内容；
7. 含有医疗机构名称、地址、联系方式、诊疗项目/方法，以及义诊、医疗咨询电话、特约门诊等医疗服务内容；
8. 法律、行政法规规定不得含有的其他内容。

> 🔍 以上 8 项措辞依据 21 号令原文（国家市场监管总局令第21号）。第16条与第11条为叠加适用：第16条为基础红线，第11条为"三品一械"细化禁令。

## C3.5 广告审查流程（申请→审查→文号）🔍
1. **申请**：广告主向**广告审查机关**（省级药监/市场监管）提交广告审查申请及素材 🔍。
2. **审查**：审查机关对内容审查，通过后方可发布；仅宣传产品名称（含通用名/商品名）的，不再审查（21号令第23条）🔍。
3. **批准文号格式**（依据《广告法》及审查办法，格式示例）💭：
   - 药品：`×药广审（视/声/文）第××××××××号`
   - 其中"视/声/文"对应**视频/音频/文字**媒介，"×"为省级行政区简称 💭（确切格式以最新审查文书规范为准）。
4. **发布**：经审通过并**向社会公开**的广告，可依法在全国范围内发布（21号令第24条）🔍。

## C3.6 新型媒介合规（直播/种草/MCN/达人）
- 广播电台、电视台、报刊、互联网信息服务提供者**不得以介绍健康、养生知识等形式变相发布**药品/医疗器械/保健食品广告（《广告法》第19条）🔍。
- **达人/MCN 须有资质**：带货主体应具备相应经营资质；处方药严禁通过大众/直播媒介发布 🔍。
- **禁止变相广告**：科普、测评、种草内容若具广告属性，须**明示"广告"**并遵守第16/18/21号令全部禁令 🔍。
- **代言人限制**：不得利用广告代言人作推荐证明（第16条第四项）🔍；医师、药师等专业人员形象亦归入 21 号令第11条第二项禁止范围 🔍。
- 接 drug-retail R6（药店广告）、drug-training T5（零售话术护栏）。
